Event Duration Easing Usage

Rules: ease-out only. No bounce, no spring. Motion should be invisible — users notice data, not animation.

Tailwind usage (L3)

Semantic tokens map directly to Tailwind utilities via @theme inline in globals.css.

CSS token Tailwind class Example
--background bg-background Page body background
--primary bg-primary / text-primary Primary button, active link
--muted-foreground text-muted-foreground Placeholder, caption text
--border border-border Applied globally via * { @apply border-border }
--radius-lg rounded-lg Cards and panel containers
Non-negotiables
Rule Details
Typeface DM Sans only. Never substitute Inter, Geist, or any other font.
Components shadcn/ui only. Never write raw HTML buttons, inputs, or selects.
Icons Lucide React only. strokeWidth 1.5. Never use filled icons or other icon sets.
Tokens Semantic tokens only in JSX. Never hard-code hex, rgb, or L1 primitive names.
Dark mode Toggle via .dark class on <body>. Both modes fully supported.
Data density Compact layout is intentional. Default body 14px, compact 12px. Don't inflate spacing.
